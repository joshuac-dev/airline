Cost per PAX (Mod)

It is recommended to download Better Airline Club instead of Cost per pax as it includes more features and uses the updated fuel formula released on September 1st, 2025. Cost per pax is no longer maintained and calculations may be highly inaccurate.

___

Cost per PAX, formerly known as Plane Score, is a mod made by Alrianne Trans Inc that shows max frequency that a plane can fly on a route, fuel burn per passenger, fuel burn per week and overall score when given route distance. The latest version of the mod is available at https://github.com/wolfnether/Airline_Club_Mod/raw/main/Cost_per_pax.user.js.

Author

Alrianne Trans Inc

Features

Shows max frequency that a plane can fly on a route and how much it costs to run the flight per pax if you were flying all economy planes.


Instructions

  1. Go to Github link ([1])
  2. Copy the paste
  3. Open Airline Club tab
  4. Press F12 (or right click/inspect element anywhere on the page)
  5. Paste mod in the console part at the bottom of the Devtools next to the ">"
  6. Press enter
  7. (in game) Click the airplane tab to see the new changes

Cost per PAX (and mods in general) can be automatically applied with userscript managers, such as Tampermonkey.

Note: The default settings for Cost per PAX automatically excludes airplanes which require 3000m+ runways, which may cause confusion. This can be fixed by editing the runway variable to 3600+ (one time only), or by changing the default runway value in row 137.
