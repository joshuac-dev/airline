Persistent Install (recommended)

  1. Download the Tampermonkey extension or similar.
  2. Click this link: https://github.com/Bohaska/bac/raw/refs/heads/main/Better%20Airline%20Club.user.js
  3. Click "install" when it asks if you want to install the script.
  4. If you are in Chrome (possibly in other browsers as well), make sure you have this setting enabled: https://www.tampermonkey.net/faq.php?locale=en#:~:text=In%20desktop%20Chrome%2FEdge%20138,Manage%20Extension"%20(2).&text=Locate%20and%20enable%20the%20"Allow%20User%20Scripts"%20toggle
  5. Restart your browser after you've completed step 4.
  6. Open https://airline-club.com and you should see all the BAC enhancements!

Temporary Install

  1. Copy the code at https://github.com/Bohaska/bac/blob/main/Better%20Airline%20Club.user.js
  2. Open Airline Club tab
  3. Press F12 (or right click/inspect element anywhere on the page)
  4. Paste mod in the console part at the bottom of the Devtools next to the ">"
  5. Press enter
  6. (in game) Click the flights tab to see the new changes

Key Changes

SF Column is changed to (SF-60%)/(100-40)%, since loyalist conversion only occurs when SF>60%.

Profit shown is that after accounting for office SQ expenses, which the default game considers an "other expense". As a result, the sum of the profits in your routes tab equals flight profit - office SQ costs in Office and not simply flight profit.

Cost-per-pax estimations in "Market" page do not account for office SQ cost, and is slightly inaccurate for discounted planes. Cost-per-pax can be inaccurate at times for various other reasons as well.

Profit margin (revenue + profit/profit), profit per passenger, profit per flight and profit per flight duration.
