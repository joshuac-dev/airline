Features

Quick mod to add a few features here and there to AC.

Meant to be compatible with Better Airline Club Mod

Installation

  1. Install Tampermonkey or Greasemonkey on either Chrome or Firefox
  2. Open this page on your browser https://github.com/mathdons/Airline-Club-Tweaks/raw/main/AirlineClubTweaks.user.js
  3. This should trigger the installation of the script
  4. Updates are automatically retrieved once per day, or you can force the update directly in Tampermonkey

Usage

All data is pulled from existing content displayed in the game.

As of V0.1.27 three main features:

  1. Display of Loyalist Delta and expected time to overtake a rival in the Airport view
    • Only rivals displayed on the Loyalist Trend table are supported
  2. Some added statistics per base on the Office view
    • The stats are created when the Flights page is visited and refreshed each subsequent visit
  3. Added filters for Plane Usage and ConditionÂ % in the Hanger view
    • The value is stored when the Office view is visited and refreshed each subsequent visit
