Airport Charm

An airport charm is a device in the code that allows the game to create varying demand for routes. An airport will have a charm if you see any of the following symbols when clicking on an airport from the world map:

-Financial Hub: These airports will attract more business type passengers.

-Gateway Airport: These airports will have extra demand to other gateway airports. More common for smaller island nations.

-International Hub: These airports will attract passengers from other countries, most likely business type passengers.

-Isolated Town: These airports will get a boost to demand to domestic routes no longer than 1,000 km and have a population of 100,000 or more.

-Vacation Hub: These airports will attract more tourist type passengers.

The greater the value next to a symbol, the "stronger" the charm is. In other words, it increases the number passengers that can be added to a route to that airport.
