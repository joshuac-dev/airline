Airline Club Teak Mod

Quick mod to add a few features here and there to AC.

Meant to be compatible with

Better Airline Club Mod

Installation

Usage

All data is pulled from existing content displayed in the game.

As of V0.1.27 three main features:

  • Only rivals displayed on the Loyalist Trend table are supported
  • The stats are created when the Flights page is visited and refreshed each subsequent visit
  • The value is stored when the Office view is visited and refreshed each subsequent visit

See

https://github.com/mathdons/Airline-Club-Tweaks

for details.
